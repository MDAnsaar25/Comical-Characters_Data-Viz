# Comical Characters Data Viz

A fun data analysis and visualization project exploring comical cartoon and animated characters.  
It turns character datasets into charts, rankings, and insights about popularity, traits, and trends.

## About the Project
This project analyzes datasets of comical characters from cartoons, movies, and TV shows.  
The goal is to uncover patterns in character traits, popularity, and representation across different series using Python and data visualization libraries.

It’s built for practice in EDA, data cleaning, and making data visually engaging.

## Features
- **Character Rankings**: Top characters by popularity, ratings, or screen time
- **Trait Analysis**: Most common personality traits, quirks, and roles
- **Show/Series Breakdown**: Compare characters across different shows and movies
- **Visual Insights**: Bar charts, pie charts, heatmaps, word clouds, and scatter plots
- **Clean Workflow**: End-to-end EDA from raw CSV to final visualizations

## Tech Stack
- **Language**: Python 3.x
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn, Plotly
- **Environment**: Jupyter Notebook / VS Code

##Installation

## Dataset
Place your dataset in the `/data` folder as `characters.csv`.  
Expected columns: `Name`, `Show`, `Trait`, `Popularity`, `Age`, `Role`, etc.  
You can use any dataset of animated/comical characters. Kaggle has a few good ones.
